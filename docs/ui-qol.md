# UI / 显示 QoL 方向

## 目标

在保留 Red Alert 视觉识别的前提下，提升现代显示器（含 macOS Retina）上的可读性与操作反馈。

## 候选改进（Week 3）

1. **主菜单 / 设置页**：对比度与字号微调，避免「糊成一片」
2. **局内 HUD**：电力不足、低资金等警告更醒目（可关）
3. **快捷信息**：选中单位时关键状态更清晰
4. **高 DPI**：记录 OpenRA 在 Mac 上的缩放行为与建议设置

## 约束

- 不做网游风大厅重做
- 不在首屏堆砌统计卡片式 UI
- 默认主题仍应一眼可辨为 RA / OpenRA
- Chrome 改动放在 `mod/ra-enhanced/chrome/`，并写明覆盖了上游哪一层

## 现状

当前仓库提供 `chrome/` 说明与占位 YAML，实装按 ROADMAP Week 3 推进。
