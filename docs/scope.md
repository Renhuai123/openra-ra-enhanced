# 项目范围（Scope）

## 我们做什么

- 基于 **OpenRA** 的 Red Alert **增强 Mod**（规则、AI、UI/QoL、文档）
- **免费、开源**（GPL-3.0），可在 macOS / Windows / Linux 上运行（依赖 OpenRA）
- 提升 AI 质量与现代显示/操作体验
- **默认保持经典 RA 手感**；增强项尽量可配置或可选

## 定位（对外一致口径）

- 抖音/社区常见需求：红警画面与 AI 体验增强 → 我们用 **OpenRA Mod** 回应
- **不能**做独立商业红警产品（IP 在 EA）
- **能**做 Mod：更好 AI、QoL、后续嵌入式 AI 玩法（敬请期待）
- 详见 [ANNOUNCEMENT.md](ANNOUNCEMENT.md)

## 我们不做什么

| 不做 | 原因 |
|------|------|
| 分发 Red Alert 游戏二进制或版权资源包 | 版权；违反 OpenRA 内容政策精神 |
| 商业重制 / 出售游戏本体 | 本项目定位为社区 Mod |
| 声称可替代购买/合法获取原版资源的义务 | 玩家须自行合法导入资源 |
| 首期大幅重做单位数值与科技树 | 会破坏「经典手感」目标 |
| 提供破解、盗版资源下载链接 | 明确禁止 |

## 资源政策（硬性）

1. 本仓库 **只** 包含 Mod 定义、脚本、文档与自有/开源素材。
2. 玩家通过 OpenRA **官方 Content Installer / Asset Import** 获取 RA 内容。
3. PR 若包含可疑的 `.mix`、原版可执行文件、完整音效/视频包等，将被拒绝。
4. Issue / README 中不引导盗版源。

## 成功标准（首期）

- 用户能在 Mac 上安装 OpenRA → 导入资源 → 加载本 Mod
- 文档清楚说明合法性边界
- AI / UI 有可感知改进且未明显破坏经典节奏
